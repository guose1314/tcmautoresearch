# AutoResearch Program

## 研究目标（中文指令入口）
请在不改变训练脚本输入输出协议的前提下，自动改进 `train.py` 的验证集 `val_bpb`（越低越好），并控制显存占用。

## 固定约束
1. 必须保持 `train.py` 可执行。
2. 每轮训练最长 300 秒。
3. 输出中必须包含：`val_bpb=...`。
4. 若结果变差或崩溃，回滚到当前基线代码。
5. 仅允许在 `train.py` 内进行改动。

## 成功判据
- `val_bpb` 下降即视为有效改进。
- 改进后自动 commit，成为新基线。

## 变更边界
- 禁止改动仓库内其他业务模块。
- 禁止破坏日志输出格式。

## 建议优化方向
- 学习率调度
- 正则化强度
- 批大小与序列长度平衡
- 梯度裁剪阈值
- 优化器超参数
